📮 Postal Operations Interface
A Desktop Application for Local Post Office Postal Management System

This project is a desktop-based Postal Management System developed using C# Windows Forms, SQL Server, and ADO.NET. It automates daily postal operations such as parcel registration, tracking, staff management, delivery status updates, and reporting.

📌 Project Overview

The system was developed to replace manual postal record management with a secure and efficient computerized solution. It enables postal staff to register parcels, calculate postage automatically, search parcel records, manage staff accounts, generate receipts, and monitor parcel statistics.

🚀 Features

✅ User Login Authentication

✅ Role-Based Access Control (Admin & Staff)

✅ Parcel Registration

✅ Automatic Postage Cost Calculation

✅ Unique Tracking ID Generation

✅ Receipt Generation

✅ Search Parcel by Tracking ID

✅ Search Parcel by Sender Name

✅ Update Delivery Status

✅ Delete Parcel Records

✅ Staff Account Management

✅ Reports & Analytics Dashboard

✅ SQL Server Database Integration
